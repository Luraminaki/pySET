# Changelog

All notable changes to this project are documented here. Versioning loosely
follows [Semantic Versioning](https://semver.org/); the project is still
`-alpha`, so breaking changes can happen between minor versions.

The current version lives in [pyproject.toml](pyproject.toml) and is read
from package metadata at runtime.

## 0.3.0-alpha

- Room visibility & password protection.
- Full frontend/backend modernization: Nuxt 4, Pinia state management,
  Pydantic-based backend config, `pyproject.toml` packaging, pytest suite.
- Playwright e2e test suite.

- Multiple Game Sessions.
